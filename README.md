# University
This repository contains solutions and projects developed during my studies at the University of Wrocław.

### Courses done:
#### 1st semester:
* Analiza matematyczna (Calculus)
* Logika dla informatyków (Logic for Computer Science)
* Wstęp do informatyki (Introduction to Computer Science)
* Wstęp do programowania w C (Introduction to Programming in C)
* Podstawowy warsztat informatyka (Tools for Computer Scientists)

#### 2nd semester:
* Metody programowania (Programming Methodology (Racket))
* Algebra
* Programowanie obiektowe (Object Oriented Programming)
* Kurs języka C++ (C++ Course)

#### 3rd semester:
* Analiza numeryczna (Numerical analysis)
* Matematyka dyskretna (Discrete Mathematics)
* Wybrane praktyki projektowania oprogramowania (Selected Aspects of Software Development)
* Kurs języka Rust (Rust Course)
* Metody implementacji algorytmów (Algorithm Implementation Methodology)

#### 4th semester:
* Algorytmy i struktury danych (Algorithms and Data Structures)
* Architektury systemów komputerowych (Computer System Architecture)
* Rachunek prawdopodobieństwa i statystyka (Probability and Statistics)
* Bazy danych (Databases)

#### 5th semester:
* Machine Learning
* Kurs projektowania aplikacji ASP.NET (ASP.NET Application Design Course)
* Kurs rozszerzony języka Python (Extended Python Course)
* Inżynieria oprogramowania (Software Engineering)
* Seminarium: implementacja interfejsów użytownika (User interface implementation seminar)
* Metody implementacji algorytmów (Algorithm Implementation Methodology)(2nd time)

#### 6th semester:
* Sieci komputerowe (Computer networks)
* Systemy komputerowe (Computer Systems)
* Projektowanie obiektowe oprogramowania (Object-Oriented Software Design)
* Kurs tworzenia aplikacji frontendowych (Frontend Application Development Course)

#### 7th semester:
* Kurs języka Java (Java Course)
* Programowanie współbieżne (Concurrent Programming)
* Kurs: Podstawowe narzędzia DevOps (DevOps Tools Course)
* Warsztat AI (AI Workshop)
* Innovative Projects by Nokia
